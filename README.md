Overview

The Medicine Tracker Web App is a full-stack application designed to help users manage their daily medication schedule efficiently.
It solves a real-world problem — people often forget whether they have taken their medicine or not. Applications like these typically provide reminders, 
logs, and tracking history to improve consistency and health management .
This project focuses on clean UI, usability, and real-time tracking, making it simple for users to manage their medications.


Features:-

:- Add & Manage Medicines

:- Add medicine name, dosage, and schedule

:- Edit or delete entries easily

:- Reminder-Based Tracking 
    (Helps users stay consistent with their medication routine)
    
:- Daily Tracking System

:- Mark medicines as taken 
    (Avoid confusion like “Did I take it or not?)
    
:- History / Logs

:- View past medication activity

:- User-Friendly UI

:- Clean and minimal interface for better usability

Tech Stack -> React js, Java, Spring boot

Database -> MongoDB Atlas

Frontend deployment -> vercel

Backend deployed -> Railway
